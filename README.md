# python_auto_test
python auto-test for lern
## Пробный проект для UI авто тестов на Python
---
## For https://playwright.dev/

**Project setup**
For Windows
```
pip install virtualenv
virtualenv venv
```
For Command Promt
```
venv\Scripts\activate

```
For PowerShell
```
.\venv\Scripts\Activate.ps1

```
```
pip install -r requirements.txt

```

Test run:

```
python -m pytest

```
